# UsuariosApi

Desenvolvimento de uma API no **ASP .NET 6** para realização de autenticação de usuários usando o **Identity**, com integração do **Entity Framework** e utilização do banco de dados **MySQL**.

A API tem como objetivo fornecer um sistema de autenticação seguro e eficiente, permitindo aos usuários realizar login e gerenciar suas informações pessoais. A implementação é baseada nas melhores práticas do ASP .NET 6, aproveitando as funcionalidades avançadas do Identity para garantir uma autenticação robusta e escalável.

O **Entity Framework** é utilizado para a interação com o banco de dados MySQL, proporcionando um mapeamento objeto-relacional eficiente e simplificado.
